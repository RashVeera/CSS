CSS - Cascading Style Sheet

Types

  • Internal - Use style tag
  • External - Use a separate css file
  • Inline - use style inside the element

Priority of css stylesheets are as follows : Inline > Internal > External. Also the styles applied in JS is Inline.

Cascading Effect

Any rule of the same css specificity will take precedence over

CSS Combinators

  • Descendant combinator (' ')
  • Child Combinator ('>')
  • Adjacent Sibling combinator ('+')
  • General Sibling Combinator('~')
